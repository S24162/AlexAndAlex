Java version 17; Spring Boot v3.5.5
Deployed on Render.com: https://alexandalex.onrender.com/
When starts from local: http://localhost:8080/
Prototype was done in Figma and can be found here: https://www.figma.com/proto/9LbxX9meOXFtSOcd7Pf7hu/Furniture?node-id=1-4&m=draw&scaling=scale-down&content-scaling=fixed&page-id=0%3A1&starting-point-node-id=19%3A3&t=G3x0B4VjmBz2oOze-1

Diploma project focused on building a web application based on Spring Web.
Project covering the full development lifecycle: 
business analysis and direct client communication; creation of a Figma prototype; 
collaboration with a front-end developer to implement the UI based on the prototype; 
back-end and database development; and deployment to Render.com using Docker.

Five-page navigation structure.
Every section, except "Ceny": rendering on server side by Thymeleaf template engine; MVC controllers.
Section "Ceny", "Oblicz" button : rendering on client side with JS; REST controller.

Multi-database support H2, MySQL, PostgreSQL (in .properties files) and Docker deployment on Render.com.
